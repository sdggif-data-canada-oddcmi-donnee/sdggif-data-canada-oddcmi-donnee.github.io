Reporting status

The reporting status tab provides an overview of how many Sustainable Development Goal indicators Canada measures and reports on. For more information, see What does the reporting status mean?

The disaggregation status tab provides an overview of how many Sustainable Development Goal indicators include data disaggregated according to the United Nations (UN) requirements for each indicator, with some exceptions. Proxy indicators that differ significantly from the UN metadata are assessed using national standards and are considered fully disaggregated when these standards are met. Proxy indicators that align closely with the UN metadata continue to be evaluated against the UN’s disaggregation requirements. Indicators with the following statuses are considered out of scope: data not available, data not applicable, and exploring data sources
